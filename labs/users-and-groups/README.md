# Lab: Users and Groups Management

## Scenario
An organization needs to manage user accounts and control access to resources. As an IT Support Specialist, I was responsible for creating users, assigning them to groups, and managing permissions.

## Objective
- Create and manage user accounts
- Assign users to appropriate groups
- Understand how group membership controls access

## Tools Used
- Windows Operating System
- Computer Management
- Local Users and Groups

## Steps Performed

### Accessed User Management
- Opened Computer Management
- Navigated to "Local Users and Groups"

### Created a New User
- Created a new user account
- Set username and password
- Configured password settings

### Created/Modified Groups
- Reviewed existing groups
- Created a new group (if required)

### Added User to Group
- Assigned the user to a group
- Verified group membership

### Verified Access
- Confirmed the user had appropriate permissions
- Tested login (if applicable)
